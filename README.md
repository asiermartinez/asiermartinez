### Hello, I'm Asier 👋

Staff Product Designer at GitHub. Over the years, I've had the opportunity to design some of GitHub Advanced Security's most impactful features: Secret Scanning Custom Patterns and Push Protection, Code Scanning Default Setup and its alert integration in pull requests, Copilot Autofix, and Security Campaigns, all focused on making security feel like a natural part of how developers build software. More recently, I've transitioned to leading design for the Copilot Metrics team, where we surface insights into how developers use Copilot to help organizations get the most out of it.
